# `<text-editor>`

## Example:

```html
<link rel=import href=https://snuggsi.es/text-editor>
<text-editor></text-editor>
```

## Browser Support

  * IE 9+
  * Chrome 5+
  * Firefox 4+
  * Safari 5+
  * Opera 11.6+
