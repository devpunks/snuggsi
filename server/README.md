## .mount

  { Server }
    = require ('snuggsi')

(new Server)
  .mount ( 'foo', resource `bar` )
  .serve  ``

## .serve

Serve a specified directory as root

```javascript
const
  { Server }
    = require ('snuggsi')

(new Server).serve  ``
```
